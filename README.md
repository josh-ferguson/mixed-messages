# Mixed Messages
This is a script that creates a random mixed message from three message components:
+ Name.
+ A primary function.
+ Emotion.

This will then create a message from an imaginary AI.

## How to Use
1. Clone repository.
2. Open Terminal or Command Prompt.
3. Set Path to where File is Located (using cd).
4. Type “node script.js” and Click Enter.

## Example Results
- "Hello I am an AI and my name is Jess. My primary function is to make the world a better place and this makes me feel confused!"
- "Hello I am an AI and my name is Joy. My primary function is to create art and this makes me feel angry!"
- "Hello I am an AI and my name is David. My primary function is to write poetry and this makes me feel regret!"
